Conhecendo a DOM

O que é a DOM?

A DOM significa Document Object Model

Traduzindo fica algo como: Modelo de Objeto de Documento

É basicamente uma representação da página web em forma de árvore, onde cada elemento HTML é um nó(ou node) que podemos acessar e manipular com o JavaScript.

Como funciona?

Quando o navegador carrega uma página, acontece o seguinte:

  1. O navegador lê o HTML e cria o DOM
  2. O JavaScript pode então navegar, alterar, criar ou remover elementos nesse DOM
  3. O navegador atualiza a página com as mudanças que você fez.

O que dá pra fazer com o DOM?

  • Selecionar elementos
  • Ler ou alterar conteúdo
  • Alterar atributos
  • Alterar estilos
  • Criar ou remover elementos
  • Brincar com eventos
